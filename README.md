#  K-BOX WEB-BASED APPLICATION 

K-BOX is a Karaoke App that allows you to sing songs from Global Top Artists. The app allows you to add your favorite songs and create a Room that allows you and other users who are in the room to add songs to sing along to. 

The App is build using Meteor as the application platform, while using React to build the UI interface and  MongoDB as a database. 

# Technologies: 

The following technologies were applied in building and delivering K-BOX's application:- 

1.	JavaScript  
1.  MongoDB
1.  Meteor
1.  React-Final-Form 
1.  Material-UI
1.  React-Router
1.  React-Youtube
1.  sweetalert2
